Spam Email Detection System

A machine learning-based project that classifies emails as Spam or Not Spam (Ham) using natural language processing (NLP) and supervised learning algorithms. The system analyzes email text content and predicts whether the message is legitimate or suspicious.

Project Overview

Spam emails are unwanted messages often used for advertising, phishing, or scams. This project aims to automatically detect spam emails by training machine learning models on a labeled dataset of email messages.

The project uses text preprocessing, feature extraction, and classification techniques to improve detection accuracy.

Features
Email text cleaning and preprocessing
Spam/Ham classification
Multiple machine learning algorithms
Model comparison for accuracy
Save and load trained models
Predict custom email messages
